# React-Bulletin-Board

_This is a small sample application to understand basics of React.js_

## How it works:

This is a bulletin board app where user can create new notes, update and delete those. The app is integrated with node and mongoDB.

## How to Run:

1. Clone the app
2. Do npm install
3. Change URL in board.js to local system ip.
4. MongoDb should be installed in system. Start Mongo. Change 'dbPort' to Mongo's port in config.js file.
4. npm start
